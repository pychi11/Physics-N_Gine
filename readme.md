# Physics N_gine

## Installation

setup an environment for python
example in conda environment:

```bash
    conda create --name physics_ngine python=3.13
```

activate environment with:

```bash
    conda activate physics_ngine
```

To install Physics N_gine, you can use pip:

```bash
    pip install pygame pygame_gui
```

or install through the requirment.txt file

```bash
    pip install requiements.txt
```

## Features

**Simulation** - Simulate physics of objects in a 2D environment.

**GUI** - Create a graphical user interface (GUI) for the simulation.

**Physics** - Gravity, bouncing/restitution
